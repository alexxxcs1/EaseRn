# EaseRn

## 目标

想要 做一个 基于 React-Native 的集成框架 

## 已经实现的功能

* 简单上手的可配置App组件

* 可配置的基础路由（rn的react-navigation 4.x 真是脑壳疼）

* 2019-10-2 增加HeadView组件 该组件用于替换默认头部
